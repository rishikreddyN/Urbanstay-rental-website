# UrbanStay 

UrbanStay is a full-stack rental and hotel booking web application inspired by Airbnb.  
Users can explore listings, create properties, upload images, and leave reviews.

## Features
- User authentication (login/signup)
- Create, edit, delete listings
- Image upload with Cloudinary
- Reviews & ratings
- MongoDB Atlas database
- Responsive UI

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- EJS
- Cloudinary
- Passport.js
- Bootstrap

## Deployment
Deployed using Render (cloud platform) for full-stack hosting.
